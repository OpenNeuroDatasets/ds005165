This is the README file for the ./derivatives/versionB folder in the BOLD Moments Dataset.

This folder contains the preprocessed data corresponding to the version B pipeline (see Supplementary for an overview).

MNI152: preprocessed data, including GLM and preparation of beta values, pertaining to the data in MNI152 volume space.

fsLR32k: preprocessed data, including GLM and preparation of beta values, pertaining to the data in fsLR32k surface space.

fsaverage: preprocessed data, including GLM and preparation of beta values, pertaining to the data in fsaverage surface space.

fmriprep: fmriprep outputs. It additionally includes BOLD time series interpolated to a TR of 1 second (*interpTR1s_bold.nii).

analysis: intermediate results corresponding to version B analyses.

scripts: the scripts used in data preparation and some analyses.
